Selectores CSS

¿Que son y para que sriven?: De acuerdo con la tabla elegí el tema de selectores CSS. ¿Qué son y para que sriven? Definen principalmente sobre qué elementos se utilizan un conjunto de reglas CSS como, por ejemplo. Sintaxis: elthame Ejemplo: input, se aplica a cualquier elemento Selectores de clase: En todos los elementos se tiene un atributo de class especificado. Sintaxis: .classname Ejemplo: .index se selecciona cualquier elemento que tenga esta clase, “index”

Tipos de selectores ¿Como se utilizan los selectores CSS? Selector de ID: Podemos seleccionar un elemento lo cual el valor de su atributo id. Solo puede haber un elemento determinado dentro de un documento. Sintexis: #idname Ejemeplo: #toc, se utiliza para cualquier elemento que tenga el ID “toc” Selector universal: Se selecciona todos los elementos. Opcionalmente esta restringido a un espacio de nombre específico o en los espacios de nombres. Sintexis: *ns l * * l *¨ Ejemplo: * se aplica a todos los elementos del documento. Otros selectores y combinadores Selector de atributos: Selecciona elementos extremadamente en el valor de un determinado. Sintaxis: [attr] [attr=value] [attr-=value] [attr l =value] [attr^=value] [attr$=value] [attr*=value] Ejemplo: [autoplay] seleccionará todos los elementos que pueden tener un atributo de “autoplay”. Combinadores: Combinador de hermanos adictivos. Los combinadores + selecciona hermanos adictivos. Esto quiere decir que el segundo elemento sigue directamente al primero y ambos comparten el mismo elemento padre. Sintaxis: A + B Ejemplo: La regla h2 + p se utilizará a todos los elementos.
